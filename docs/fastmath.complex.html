fastmath.complex

Complex numbers functions.

Complex number is represented as Vec2 type (from clojure2d.math.vector namespace).

To create complex number use complex, vec2 or ->Vec2.

Simplified implementation based on Apache Commons Math. Functions donâ€™t check NaNs or INF values.

Categories

  • Power / logarithm: exp log pow
  • Trigonometry: acos asin atan cos cosh csc sec sin sinh tan tanh

Other vars: abs add arg complex conjugate div mult neg reciprocal sq sqrt sqrt1z sub

Constants

  • I = #vec2 [0.0, 1.0]
  • I- = #vec2 [0.0, -1.0]
  • ONE = #vec2 [1.0, 0.0]
  • TWO = #vec2 [2.0, 0.0]
  • ZERO = #vec2 [0.0, 0.0]
